Question-2: What is the difference between innerHTML, innerText, and textContent ?


"innerHTML" - retrieves the complete HTML markup within an element, including tags
"innerText" - only returns the visible text content of an element, ignoring HTML tags
"textContent" - returns all text content within an element, including hidden text, but without considering HTML formatting

Question-3: What is event delegation in the DOM?


Event delegation is a kind of js technique where we can attach single event listener to a parent element....What helps us to reduce the number of event listeners..

Question-4: What is event bubbling in the DOM?


Event bubbling is a mechanism what allows parent elements to respond to events triggered by their child elements.
